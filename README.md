# Java-Food-Orders


Android Application of a food ordering app giving the users options for meals.

## Description
This project uses unique and robust interactive elements to make the user exeperience pleasing as well as insightful for data. This project uses rating stars, checkboxes, radiogroups, 
spinners and switches


## Features
- Rate the order
- Price calculation


## Technologies
- Language / Framework: Java


## How to Run
1. Clone this repository.
2. Open the folder in **Android Studio**.
3. Let Gradle sync and build automatically.
4. Click **Run ▶️** to install on a device or emulator.
